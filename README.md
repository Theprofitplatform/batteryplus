# Batteries Plus NSW

Professional website for Batteries Plus NSW — car, truck, marine, and emergency vehicle battery services.

## Overview

A multi-page business website featuring service information, product listings, and contact forms for a NSW-based battery specialist.

## Tech Stack

- **Frontend:** HTML5, Tailwind CSS, Vanilla JavaScript
- **Build Tool:** Vite 5
- **Testing:** Playwright (accessibility), Lighthouse (performance)
- **Image Optimization:** Sharp, Imagemin
- **Icons:** Font Awesome

## Pages

- `index.html` — Homepage with services overview
- `services.html` — Detailed service listings
- `products.html` — Battery product catalogue
- `about.html` — Company information
- `contact.html` — Contact form and details

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Run tests
npm test
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start Vite dev server |
| `npm run build` | Optimize images + build |
| `npm run preview` | Preview production build |
| `npm test` | Run Lighthouse + accessibility tests |
| `npm run optimize:images` | Optimize images with Sharp |
